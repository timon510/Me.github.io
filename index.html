Web Application

farmers Coperative Input Rationing System

This project was developed using css, java script, php, sql, html:5. This project was buit with the aim of addressing input (seed) distributions in farmers coperatives. This solution enables for the tracking of who and how distributions are being made, when they are being made and the quantity being distributed. It also keeps a record of suppliers. apart from this, it also keeps an updated record of the condition of seeds i.e expired or not.
